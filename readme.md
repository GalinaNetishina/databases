# Парсер результатов торгов в БД

Заполнить .env и загрузить данные
```bash
python databases/task2/main.py
```

Для проверки запустить сервер
```bash
uvicorn databases/task2/main:app 
```
[просмотр](http://127.0.0.1:8000/docs#/)
